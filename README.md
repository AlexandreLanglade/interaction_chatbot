# BE Intégration - Mineure Interaction Avancée

## Requirements :
- Python 3.6.0
- RASA
```
pip3 install rasa==2.8
```
- SpeechRecognition 3.8.1
```
pip install SpeechRecognition
```
- ivy-python
```
pip install ivy-python
```
- pygame
```
pip install pygame
```
- python text to speech
```
pip install pyttsx3
```
